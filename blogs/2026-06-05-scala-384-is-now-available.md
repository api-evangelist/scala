Scala 3.8.4 has been released, bringing the latest patch updates to the Scala 3 compiler. This release continues the ongoing development of the Scala language maintained by EPFL and the broader Scala community. Users are encouraged to upgrade to take advantage of bug fixes and improvements included in this version.
